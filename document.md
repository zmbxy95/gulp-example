## Gulp工作流说明
### Gulp需要处理的内容
- HTML文件
  - 引用叫文件的处理
  - 模板语法的处理
  - 文件压缩处理
- js文件
  - 新特性语法的Babel转换
  - 文件压缩处理
- 样式文件处理文件
  - less/sass文件的处理
  - 文件压缩处理
- 图片
  - 文件压缩处理
- 字体图标处理
  - 文件压缩处理

### Gulp 任务使用

- 按照任务需求，对每一类需要进行处理的文件编写对应的任务，然后将对应的文件输出到目标文件。
- 对不需要的文件，直接进行流读写输出到目标目录。
- 对HTML文件处理，需要注意文件引用的处理以及引用文件的压缩
- 将上述任务编写后，按照开发环境和生产环境，增加开发环境任务与生产环境任务，并对任务进行导出，暴露到外部供使用。
- 开发环境：编译 + 运行server
- 生产环境：清理文件 + 编译 等一系列处理。
- 任何组合上，文件处理任务，对不同类型的文件可并行处理。
- 对最终执行的任务应当按照（清理）编译、运行的顺序序列执行。

## Gulp 使用方式

```shell
# 文件清理
yarn gulp clean
# 运行开发环境
yarn gulp serve
# 生产编译
yarn gulp build
```
